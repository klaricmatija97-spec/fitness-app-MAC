# Video Folder

Stavite vaš motivacijski video iz teretane ovdje.

## Naziv fajla:
- `gym-motivational.mp4`

## Format:
- MP4 format
- Preporučena rezolucija: 1920x1080 ili viša
- Preporučeni codec: H.264

## Napomena:
Ako video fajl ne postoji, aplikacija će automatski koristiti fallback video s Pexels-a.
